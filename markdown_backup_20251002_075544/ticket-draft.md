# /ticket-draft

Save the current ticket to the drafts directory with the standard naming.

## Usage

```bash
/ticket-draft
```

## Behavior

- Path: `linear/tickets/drafts/{TEAM}-{slug}-draft.md`
- Operates on current draft by default
- Asks before writing if file exists
